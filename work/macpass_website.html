MacPass Landing Page

MacPass is basically a free, open-source, KeePass-compatible password manager for macOS.

My approach was to keep the website as short as possible, so I added the "But there is more..."-section to the bottom of the website to show the user all features that are worth to highlight, but in the shortest possible way.
